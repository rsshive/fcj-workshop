## My AWS First Cloud Journey - 12-Week Internship Worklog

During my 3-month internship at **AWS First Cloud Journey** (September 9 - December 9, 2025), I completed a comprehensive cloud computing training program focused on AWS services and practical implementations. This worklog documents my weekly progress, learning objectives, and hands-on experiences throughout the program.

The internship was structured around progressive learning, starting from fundamental AWS concepts and gradually advancing to complex cloud architecture patterns, networking solutions, and security best practices. Each week built upon previous knowledge while introducing new services and real-world scenarios.

### Weekly Overview:

**Week 1:** [Getting Started with AWS - Account Setup & Basic Services](1.1-week1/)  
Initial setup, AWS account configuration, IAM fundamentals, and introduction to core services.

**Week 2:** [AWS Identity & Access Management (IAM) and VPC Fundamentals](1.2-week2/)  
Deep dive into IAM policies, roles, and Amazon VPC networking essentials.

**Week 3:** [Amazon EC2 and Compute Services](1.3-week3/)  
Launching and managing EC2 instances, understanding compute options and pricing models.

**Week 4:** [Storage Services - S3, EBS, and EFS](1.4-week4/)  
Exploring AWS storage solutions, S3 bucket configurations, and file system options.

**Week 5:** [Databases on AWS - RDS, DynamoDB, and More](1.5-week5/)  
Working with managed database services and understanding different database types.

**Week 6:** [Networking Deep Dive - VPC Advanced Concepts](1.6-week6/)  
Advanced VPC configurations, subnets, route tables, and network security.

**Week 7:** [Security and Compliance Best Practices](1.7-week7/)  
Implementing security groups, NACLs, encryption, and AWS security services.

**Week 8:** [Serverless Computing - Lambda and API Gateway](1.8-week8/)  
Building serverless applications and understanding event-driven architectures.

**Week 9:** [Monitoring and Optimization - CloudWatch and Cost Management](1.9-week9/)  
Setting up monitoring, logging, alarms, and optimizing AWS costs.

**Week 10:** [High Availability and Disaster Recovery](1.10-week10/)  
Designing resilient architectures with multi-AZ deployments and backup strategies.

**Week 11:** [DevOps on AWS - CI/CD and Infrastructure as Code](1.11-week11/)  
Automating deployments with CodePipeline, CloudFormation, and best practices.

**Week 12:** [Final Project and Workshop Development](1.12-week12/)  
Completing comprehensive S3 VPC endpoint workshop and internship documentation.
